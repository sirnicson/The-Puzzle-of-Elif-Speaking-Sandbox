🧩 The Puzzle of Elif: Speaking Sandbox

An interactive, character-driven English speaking sandbox game designed specifically to help A2-B1 language learners build speaking confidence, practice vocabulary, and relate language back to their own lives.

👉 Click here to play the live game! - https://sirnicson.github.io/The-Puzzle-of-Elif-Speaking-Sandbox/

📌 Repository Description (For the GitHub "About" Sidebar)

An interactive, character-driven English speaking sandbox game designed for A2-B1 language learners. Built with HTML, Tailwind CSS, and Web Audio.

🗺️ What is "The Puzzle of Elif"?

Instead of a standard grammar quiz or traditional vocabulary matching game, The Puzzle of Elif is a collaborative storytelling sandbox. Students are presented with 20 visual tiles (items, routines, and objects) belonging to a fictional woman in Istanbul named Elif.

The student's task is to categorize these tiles into four distinct aspects of her life. Rather than testing "correct" answers, the game acts as a springboard for low-stakes, high-output conversation.

🏠 The 4 Slots (Daily Life Categories)

To keep vocabulary concrete and accessible, the game is organized into:

Daily Life (Günlük Yaşam): Routines, habits, and work.

Free Time (Boş Zaman): Hobbies, passions, and relaxation.

The Secret (Gizli Sır): Hidden quirks, funny habits, or mysteries.

The Dream (Hayal): Future goals, plans, and wishes.

🎯 Pedagogical Features

3D Flippable Riddle Cards: Each card features a short, cognitive riddle on the back (e.g., "Sipping and typing..." or "Stamps of tomorrow") to encourage active reading comprehension and deduction.

Dual-Language Support: Slot names and item titles include subtle, italicized Turkish translations to lower anxiety and prevent students from freezing over unfamiliar words.

Slot + Piece Speaking Prompts: When a tile is correctly matched, a dynamic speaking HUD presents two questions:

About Elif: Encourages students to speculate and use their imagination about Elif's life ("Why does she work with tea? Is she busy?").

About the Student: A highly simplified, direct bridge question connecting the theme to the student's personal life ("Do you drink tea or coffee when you work?").

Gamified Mechanics: Built-in success chimes and error buzzes (generated using Web Audio synth) along with visual shake animations provide instant, satisfying game loop feedback.

🛠️ Tech Stack & Architecture

This project is built to be lightweight, modern, and dependency-free:

HTML5 & Vanilla JavaScript: Core gameplay logic, DOM manipulation, and state management.

Tailwind CSS: Responsive utility-first framework for visual structure.

Web Audio API: Real-time synthesis of sound effects (no external audio assets required).

FontAwesome: Scalable icons for a modern interface.
